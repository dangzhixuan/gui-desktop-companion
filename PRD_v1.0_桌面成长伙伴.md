# 桌面成长伙伴「晷」PRD v1.0

## 1. 产品定位

「晷」是面向个人长期使用的 Windows 桌面任务与复盘伙伴。核心价值不是维护复杂
项目，而是通过具体任务提醒、人格化问责和每日复盘，帮助用户减少遗忘与拖延。

## 2. 核心闭环

1. 用户在今日便签写下任务。
2. 角色定时读取数据库，逐项提醒未完成任务。
3. 用户通过勾选记录完成状态。
4. 晚间保存总结与反思。
5. 可选 AI 根据任务与近期记录生成点评和明日规划。
6. 明日规划经用户确认后安排到次日。

## 3. 当前功能范围

### 任务

- 今日任务新增、完成、恢复。
- 任务具有独立计划日期，未来任务不会提前提醒。
- 同名日常任务可跨天重复创建。
- 任务顾问建议可编辑后安排到明日。

### 提醒与角色

- 启动问候、周期任务提醒、晚间复盘提醒和深夜休息提醒。
- 提醒直接点名具体未完成任务。
- 严师、温柔陪伴、毒舌挚友三种人格。
- 静态角色立绘、表情、桌面拖动、缩放、收起与托盘驻留。
- 佳句按完整轮次播放，一轮内不重复。

### 复盘

- 保存每日总结、任务结果和 AI 明日规划。
- 历史记录查看与连续复盘天数。
- 全量复盘导出为 Markdown。
- 运行中一致性数据库备份。

### 隐私与系统

- AI 可完全关闭；关闭后仅运行本地任务与提醒功能。
- 启用 AI 时，界面明确说明会发送任务、近期总结和完成状态。
- API Key 优先从环境变量读取，本地配置不纳入版本控制。
- 支持 Windows 登录后自动启动。
- 关闭管理中心仅隐藏到托盘，托盘“退出”才结束程序。

## 4. 数据与隐私

任务、复盘和设置默认保存在用户本机。数据库路径为
`%USERPROFILE%\.desktop_companion\companion.db`。

AI 开启时，仅为任务建议与复盘反馈发送必要文本到 DeepSeek。AI 关闭时不进行任何
模型请求。用户可随时备份数据库或导出可读的 Markdown。

## 5. 成功指标

- 连续复盘达到 14 天。
- 记录任务后的完成率提升。
- 晚间提醒后的复盘完成率达到 70%。
- 用户能够从备份或 Markdown 中完整取回长期记录。

## 6. 非目标

- 多用户、云同步和移动端。
- 团队项目管理。
- 邮箱、日历等第三方服务集成。
- 复杂角色养成与付费系统。

## 7. 发布验收

- 全量自动化测试通过。
- 安装版可启动、托盘驻留和退出。
- 开机自启可开启与撤销。
- AI 关闭时任务和复盘内容不触发网络请求。
- 数据库备份可重新打开，Markdown 导出内容完整。
