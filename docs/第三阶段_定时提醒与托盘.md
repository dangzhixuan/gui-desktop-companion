# 第三阶段：定时提醒与系统托盘

这一阶段补齐 PRD 中 MVP 尚未落地的定时闭环：

- F-4.1：每晚到点自动打开晚间复盘页。
- F-5.1：早、午、晚按设置时间自动生成问候。
- F-7.1：保存时间设置后，调度立即按新时间重载。

## 使用方式

应用启动后会常驻运行。系统托盘菜单提供：

- 显示主窗口；
- 隐藏到托盘；
- 退出。

单击托盘图标也可以在显示和隐藏之间切换。右上角关闭按钮始终是真正退出，
不会因为后台 AI 请求或调度任务而阻止关闭。

## 到点行为

- 早安 / 午安 / 晚安：后台生成对应时段问候，并发送系统托盘通知。
- 晚间总结：显示主窗口、切换到“晚间复盘”页，并刷新当天待复盘任务。

调度由 APScheduler 执行，事件通过 Qt Signal 返回主线程，避免后台线程直接操作界面。
