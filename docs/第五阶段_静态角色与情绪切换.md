# 第五阶段：静态角色与情绪切换

角色形象使用用户提供的陆文昔三张情绪图片，并制作成透明 PNG：

- `luwenxi_smile.png`：问候、完成任务、提交复盘；
- `luwenxi_serious.png`：询问作业完成情况、晚间复盘提醒；
- `luwenxi_angry.png`：复盘提醒后 5 分钟仍未提交时。

## 桌面交互

- 角色窗口透明、置顶并可拖动；
- 双击角色可打开主窗口；
- 托盘菜单可以显示或隐藏角色；
- 角色位置会在本机保存；
- 主窗口退出时角色窗口同时退出。

这套窗口与情绪状态是 Live2D 之前的静态原型。以后替换渲染层时，问候、追问和
情绪切换业务逻辑可以继续复用。
